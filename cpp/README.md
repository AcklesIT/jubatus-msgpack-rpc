MessagePack-RPC for C++
=======================


## Requirements

Following programs are requred to build:

  - gcc >= 4.1 with C++ support
  - [msgpack](http://msgpack.sourceforge.net/) >= 0.5.0
  - [mpio](http://github.com/frsyuki/mpio) >= 0.3.3
